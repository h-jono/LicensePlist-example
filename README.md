# LicensePlist-example

* [LicensePlist](https://github.com/mono0926/LicensePlist)を設定してLicenseを生成させる.
* 例としてリッチなアニメーションを表示できるライブラリ[Lottie](https://github.com/airbnb/lottie-ios)を導入.
* CocoaPodsでライブラリ管理
